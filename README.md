# WEBSITE GIỚI THIỆU VÀ BÁN SÁCH TRỰC TUYẾN

## Giới thiệu đề tài

Đây là dự án xây dựng website giới thiệu và bán sách trực tuyến được phát triển bằng PHP và MySQL.  
Website hỗ trợ người dùng tìm kiếm, xem thông tin sách, thêm vào giỏ hàng và đặt hàng trực tuyến.  

Ngoài ra, hệ thống còn hỗ trợ quản trị viên quản lý:
- Dashboard
- Sách
- Thể loại
- Đơn hàng

Dự án được thực hiện phục vụ cho môn học / đề tài thực tập chuyên ngành Công nghệ Thông tin.
